How It Works ?

Step 1: Login/Signup

Signup or login into preferred account.

Step 2: Select Location

Select the city where crop is going to be cultivated.

Step 3: If NPK sensor available

Select Yes in the Soil tab and put NPK values.

Step 4: If NPK sensor is not available

Select No in the Soil tab and select soil type from drop down menu.

Step 5: Predict

After Clicking on submit button it will take you to dashboard for crop prediction.
